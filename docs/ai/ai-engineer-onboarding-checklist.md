# AI Engineer / Analyst Onboarding Checklist

## Неделя 1
- [ ] Доступы: репозитории, модельный registry, feature store, dashboards.
- [ ] Изучить Responsible AI Policy и Safe Prompts Checklist.
- [ ] Пройти демо UX-интерфейсов (коуч, рекомендации, i18n переключатель).
- [ ] Настроить локальное окружение inference + smoke-тесты.

## Неделя 2
- [ ] Разобрать AI Handbook: отладка матчинга, деплой моделей.
- [ ] Пройтись по incident playbook + участвовать в тренировочном drill.
- [ ] Изучить UX copy library и правила локализации (uz/ru).
- [ ] Выполнить мини-задачу: обновить промпт/фичу и оформить MR.

## Неделя 3
- [ ] Присоединиться к реальному эксперименту: прочитать план, настроить метрики.
- [ ] Пройти ревью логов explainability и заполнить sample rollout report.
- [ ] Встретиться с Legal/Trust & Safety для вопросов по фильтрам.

## Checkpoint
- [ ] Представить summary менеджеру: чему научился, какие процессы понятны, где нужны улучшения.
- [ ] Зафиксировать доступность on-call графика и buddy.
