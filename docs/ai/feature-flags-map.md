# Карта AI-фичефлагов

| Флаг | Что включает | Dev | Stage | Prod |
| --- | --- | --- | --- | --- |
| `ai.match` | smart-matching, подсказки ранжирования | ON | % rollout | OFF |
| `ai.coach` | AI-коуч отклика (текст + ставка + ETA) | ON | % rollout | OFF |
| `ai.scam_filter` | фильтр scam/spam/policy | ON | ON | % rollout |
| `ai.dispute_triage` | подсказки модераторам по спорам | ON | % rollout | OFF |
| `ai.search` | семантический поиск | ON | % rollout | OFF |
| `ai.summaries` | AI-резюме карточек | ON | % rollout | OFF |

## Дефолты
- **Dev**: все фичи включены для QA и нагрузочных тестов.
- **Stage**: включаем по флагу для пилота и dogfooding; допускается %-роллаут через ENV.
- **Prod**: дефолт OFF, включаем по очереди (scam_filter может быть включён раньше остальных).

## Быстрый отключатель
- Все флаги управляются через переменные окружения `FEATURE_AI_*`. Изменение переменной + reload.
- ai-gateway поддерживает fallback в коде: при выключении флага возвращаем дефолтный результат (обычный поиск, отсутствие подсказки).
- Для аварийного отключения: удалить ключ маршрута в сервисе ai-gateway через toggles config или выставить WAF-rule, чтобы обнулить вызовы /match и т.д.
