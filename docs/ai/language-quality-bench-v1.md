# Language Quality Bench v1

## Мультилингвальные эмбеддинги
- Базовая модель: `intfloat/multilingual-e5-large` (ru/uz поддержка).
- Пара резервных: `sentence-transformers/paraphrase-multilingual-mpnet-base-v2`.
- Для генерации: `Nous-Hermes-2-Mistral-7B` (локально) + fallback `openai/gpt-4o-mini` за фичефлагом.

## Sanity-тесты семантики
| Запрос (uz) | Ожидаемый матч (ru/uz) | Примечание |
| --- | --- | --- |
| "mobil ilovani ishlab chiqish" | "разработка мобильного приложения" | cosine ≥ 0.65 |
| "to'lov tizimi integratsiyasi" | "интеграция платежной системы" | cosine ≥ 0.65 |
| "qadoqlash dizayni" | "дизайн упаковки" | cosine ≥ 0.60 |

Если cosine ниже порога, выполняем pivot: перевод uz→ru (NLLB) → эмбеддинг ru → сопоставление.

## Тон и стиль подсказок
- Деловой, корректный, без громких обещаний.
- Короткие абзацы (1–3 предложения), избегаем канцелярита.
- Не даём юридических/финансовых советов; для таких запросов ссылаемся на правила площадки.
- Прямо обозначаем роль: «AI-коуч предлагает…», пользователь всегда может отредактировать.
