# Матрица прав сотрудников

| Действие | Moderator | Finance |
|----------|-----------|---------|
| Просмотр всех чатов/жалоб | ✅ | ✅ (read-only) |
| Создание / обновление ChatModerationCase | ✅ | ❌ |
| Применение `send_block` / `shadow_ban` | ✅ | ❌ |
| Просмотр доказательств спора | ✅ | ✅ |
| Изменение статуса DisputeCase | ✅ | ⚠️ (только `resolution_proposed`) |
| Исполнение escrow исходов | ❌ | ✅ |
| Просмотр финансовых проводок | ❌ | ✅ |
| Добавление staff заметок (audit) | ✅ | ✅ |

Легенда: ✅ — полные права, ⚠️ — ограничено сценариями (finance может переводить спор в `resolution_proposed` для синхронизации с payout), ❌ — нет доступа.

Все действия проходят через `StaffActionLog` с фиксацией пользователя, контента и timestamp.
